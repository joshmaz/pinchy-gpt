
# Security policy

If you discover a security vulnerability in this repository, please report it by emailing
`security@pinchy.org` with a clear description, reproduction steps, and any proof-of-concept
code. Do not open a public issue for sensitive security reports.

There is no guaranteed SLA for vulnerability responses. We will triage and respond as quickly
as possible, but please assume no fixed response time.

Scope: This policy applies only to the code and content in this repository (pinchy-gpt). If
your report concerns third-party services or infrastructure outside this repository, please
contact the relevant vendor or administrator.

Thank you for helping keep the project secure.
